# HSI Compressor — обзор

**HSI Compressor** — инструмент командной строки для сжатия гиперспектральных
изображений (HSI, Hyperspectral Imaging) оригинальным алгоритмом на основе
квантования спектральных эталонов (КЭКМ).

Гиперспектральное изображение — не обычная фотография: каждый пиксель содержит
сотни значений (спектральных полос), описывающих отражение света на разных
длинах волн. Такие данные применяются в дистанционном зондировании Земли,
медицинской диагностике, пищевой и фармацевтической промышленности,
геологоразведке.

## Как это работает в двух словах

Пиксели одного типа поверхности (трава, асфальт, вода) дают похожие спектры.
Вместо хранения каждого пикселя целиком программа строит **словарь эталонов** и
заменяет каждый пиксель **ссылкой на ближайший эталон** плюс компактной поправкой
восстановления. Эталоны организованы двухуровнево (классы → детали внутри
класса), а пороги сравнения адаптируются под энергию каждого пикселя.

## Быстрый старт

```bash
# Сборка
cmake -S . -B .build && cmake --build .build

# Сжатие
./hsi_compressor compress data/scene.hdr data/scene.gsd --method 4 --me 3.0 --ae 1.0

# Восстановление .dat из файлов сжатия
./hsi_compressor decompress data/scene.hdr \
    --std standarts.gsd --img image.gsd --dat-out out.dat
```

Подробности — в [USAGE.md](USAGE.md).
